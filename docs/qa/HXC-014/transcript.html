Captured evidence — real bugs fixed (anti-bluff)

The chaos suites surfaced 29 real production bugs (full per-bug detail + file:line in the docs/Issues.md HXC-014 row and the per-batch commit bodies). Defect classes:

  • Panic-in-goroutine, no recover() (process-wide crash): event, hooks, mcp×2, notification, monitoring, commands, security, template, focus, persistence.
