CG3 — install.sh

install.sh is idempotent: reads the pinned version from codegraph.version, prechecks Node.js (>=18 <25) + npm, runs npm install --prefix tools/codegraph, resolves the binary to an absolute path, runs a codegraph --version self-check.

Version pin correction. The plan pinned 0.7.11, but that version does not exist on npm (npm error code ETARGET — No matching version found for @colbymchenry/codegraph@0.7.11). See cg3-install-attempt1-fail.txt. npm view reports the available set ends at 0.7.12; 0.7.11 was never published / was unpublished. Per incorporation-plan §8.4 (version drift is an expected, reviewed change) the pin was bumped to 0.7.12 — the current latest, still inside the supported Node window (engines: node >=18 <25).
