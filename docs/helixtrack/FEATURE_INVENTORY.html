  • Framework: Angular 19 + Angular Material
  • Theme: Lime Green #BCE63B, Teal #7AA590, Mint #B2E3C2
  • Features:
    • Responsive design
    • Dark/light theme
    • Keyboard shortcuts
    • Drag-and-drop
    • Real-time updates (WebSocket)

Desktop Client (Tauri + Angular)

  • Framework: Tauri + Angular

iOS Client (SwiftUI)
