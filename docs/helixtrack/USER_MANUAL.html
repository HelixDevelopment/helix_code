HelixTrack User Manual — 2026-06-21

Overview

HelixTrack is a modern, open-source project management and issue tracking system.

Getting Started

First Login

  1. Open HelixTrack in your browser
  2. Click “Register” to create an account
  3. Fill in your details (username, email, password)
  4. Click “Register” to complete
  5. Login with your credentials

Dashboard

After login, you’ll see the dashboard with: - Recent projects - Your assigned tickets - Activity feed - Quick actions

Projects

Creating a Project

  1. Click “Projects” in the sidebar
  2. Click “Create Project”
  3. Fill in project details:
    • Name: Project name
    • Key: Short identifier (e.g., “PROJ”)
    • Description: Project description
    • Lead: Project lead
  4. Click “Create”

Managing Projects

  • View: Click on project name
  • Edit: Click “Edit” button
  • Delete: Click “Delete” button (requires confirmation)

Tickets

Creating a Ticket

  1. Navigate to a project
  2. Click “Create Ticket”
  3. Fill in ticket details:
    • Title: Ticket title
    • Description: Detailed description
    • Type: Bug, Feature, Task, Epic, Story
    • Priority: Lowest, Low, Medium, High, Highest
    • Assignee: Team member
  4. Click “Create”

Ticket Workflow

  1. Open — Newly created
  2. In Progress — Being worked on
  3. Review — Under review
  4. Testing — Being tested
  5. Completed — Done
  6. Blocked — Waiting on dependency

Ticket Actions

  • Assign: Click “Assign” and select team member
  • Comment: Add comments in the comment section
  • Attach: Upload files and attachments
  • Link: Link related tickets

Boards

Kanban Board

  1. Click “Boards” in the sidebar
  2. Select a board
  3. Drag and drop tickets between columns
  4. Filter by assignee, priority, or type

Board Columns

  • Backlog: Not yet started
  • To Do: Ready to start
  • In Progress: Being worked on
  • Review: Under review
  • Done: Completed

Chat

Starting a Chat

  1. Click “Chat” in the sidebar
  2. Select a channel or direct message
  3. Type your message
  4. Press Enter to send

Chat Features

  • Emoji: Click emoji button
  • Attachments: Click attachment button
  • Reactions: Click on message to react
  • Threads: Reply to specific messages

Documents

Creating Documents

  1. Click “Documents” in the sidebar
  2. Click “Create Document”
  3. Enter title and content
  4. Click “Save”

Document Features

  • Markdown: Write in Markdown
  • Export: Export to PDF, HTML, DOCX
  • Share: Share with team members
  • Version History: Track changes

Reports

Viewing Reports

  1. Click “Reports” in the sidebar
  2. Select report type:
    • Ticket statistics
    • Team performance
    • Project progress
    • Time tracking

Custom Reports

  1. Click “Create Report”
  2. Select metrics
  3. Configure filters
  4. Save report

Settings

User Settings

  1. Click “Settings” in the sidebar
  2. Update profile:
    • Avatar
    • Name
    • Email
    • Password
  3. Configure notifications
  4. Set preferences

Project Settings

  1. Navigate to project
  2. Click “Settings”
  3. Configure:
    • Members
    • Workflows
    • Custom fields
    • Permissions

Keyboard Shortcuts

Shortcut Action
Ctrl+N Create new ticket
Ctrl+S Save current form
Ctrl+F Search
Esc Close dialog
