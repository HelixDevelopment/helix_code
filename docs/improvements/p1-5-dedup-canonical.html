Dependency edges (read these BEFORE WP3)

  • mcp_servers TBD blocks any consumer code that imports MCP servers (CONST-040 capability flags) until canonical chosen.

Rollback per dedup

For each canonical decision, the rollback is:

cd <parent-of-removed-path>
git submodule add -f <url-from-captured-snapshot> <removed-path>
git submodule update --init --recursive <removed-path>

URLs are captured in docs/improvements/p1-5-snapshot-pre.md and the verbatim .gitmodules content snapshots referenced from there.

  • Whether Toolkit/Toolkit/Chutes (a dup-path-typo seen in HelixAgent .gitmodules) is fixed in WP3 or in WP2 restructuring — owned by WP2.
