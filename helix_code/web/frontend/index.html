Sign in

The generation endpoints are auth-gated (they drive real, paid providers). Sign in via POST /api/v1/auth/login to mint a JWT — it is held in this tab's session and sent as Authorization: Bearer … on every generate / specify call.

Generate

Sends a real request to POST /api/v1/llm/generate (or /api/v1/llm/stream). The server constructs a real provider and returns the model's actual output — no simulation.
